# mattoofahad.github.io

Personal academic website for **Fahad Mattoo** (Fahad ul Hassan Asif Mattoo).

## Overview

This is a static personal website hosted on GitHub Pages. It serves as an academic portfolio showcasing research, publications, and professional background.

## Structure

- `index.html` — Main website page
- `stylesheet.css` — Site styles
- `images_/` — Images and favicon assets
- `data_/` — Data files (e.g., PDFs, documents)
- `photos/` — Photo assets
- `Archive/` — Archived content

## Hosting

The site is live at: [mattoofahad.github.io](https://mattoofahad.github.io)

## Local Development

To preview locally, open `index.html` in a browser or serve it with any static file server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
